GIT & OTHER STUFF

otool -tv <executable_name> (prints out machine code of the program)
kill -l | list what's available
ulimit -s | check stack size in kb
uname -m | system arch
lsblk | show info on all the block devices connected to your system
cat /etc/os-release | show OS info
which <filename> | show path
type -a | show location

FILE PERMISSIONS
chmod u|g|o[rwx|rwx|rwx] <filename>
chmod 700 | rwx to user
chmod 777 | rwx to all
chmod +x <| exec

GIT COMMANDS:
git ls-files | list files in git (-c cached, -d deleted, -m modified, -o others)
git remote add <nickname> <url>
git remote -v | show remote repositories
git remote rm <reponame> | remove git remote
git clone --recurse-submodules <url> | add submodule repo to repository
